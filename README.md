## Comandos para correr el proyecto


<h1>Backend</h1>

- Run [java -jar dwf-api-2.0.5.jar] in "Flowermart/api" 

<h1>Frontend</h1>

- Run [npm install] in "Flowermart/flowermart"
- Run [ng serve] in "Flowermart/flowermart"
